# ModernTech E-Commerce Website

A simple, modern, and high-performance e-commerce website built with vanilla web technologies.

## Tech Stack
- **Frontend**: HTML5, CSS3, Vanilla JS
- **Backend**: Core PHP
- **Database**: MySQL

## Features
- Dynamic product listing and details
- Shopping cart utilizing PHP sessions
- Modern, responsive UI with CSS Grid and Flexbox
- No bloated frameworks - fast, lean, and highly optimized

## Quick Start (with XAMPP)
1. Move this repository folder to your XAMPP `htdocs` directory (e.g., `C:\xampp\htdocs\` or `/opt/lampp/htdocs/`).
2. Start the **Apache** and **MySQL** modules from the XAMPP control panel.
3. Open `http://localhost/phpmyadmin` and import the `database.sql` file. This automatically creates the database and populates dummy products.
4. Visit `http://localhost/E-commerce-website/` in your browser.
